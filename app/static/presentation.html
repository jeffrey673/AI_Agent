사용 가이드

📝 시작하기

  • 🌐 접속: http://172.16.1.250:3000
  • 🔐 회원가입: 이름 입력 → AD 자동 매칭 → 부서 선택 → 비밀번호 설정
  • 💬 질문: 자연어로 무엇이든 입력하면 AI가 자동 분류
  • 📊 결과: 표, 차트, 인사이트 자동 생성
  • 🔄 후속: 대화 맥락 유지하며 심화 분석 가능

※ 사내 AD 등록 사원만 가입 가능
※ 비밀번호 분실 시 DB팀 임재필에 초기화 요청

향후 로드맵 — 데이터 확장

📊 데이터 학습 확장

  • Phase 1 ERP DB 연동 — 재고, 발주, 물류, 원가 데이터 실시간 학습
  • Phase 2 리뷰 데이터 업데이트 — 신규 채널 리뷰 수집 자동화 + 감성 분석 고도화
